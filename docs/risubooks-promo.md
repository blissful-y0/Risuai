# 📚 리수북스 (RisuBooks) — AI 채팅을 소설처럼 읽자

## 이게 뭔데?

리수AI 채팅을 **진짜 이북리더기처럼** 읽을 수 있는 플러그인입니다.
채팅창 스크롤 노가다 대신, 페이지 넘기면서 소설 읽듯이.

---

## 주요 기능

### 📖 Kindle 스타일 리더
- 화면 터치/클릭으로 페이지 넘김
- UI 다 숨기고 **본문만** 보여줌
- 화면 가운데 탭하면 컨트롤 나타남
- 데스크톱 2단, 모바일 1단 자동 전환

### 🎨 테마 8종 + 커스텀
기본 / 다크 / 빈티지 / 그레이 / 그린 / 네이비 / 로제 / 미드나잇
마음에 안 들면 커스텀 색상 직접 만들기 가능

### 🔤 폰트 5종 + 커스텀 CSS
- 부크크명조 / 마루부리 / 리디바탕 / KoPub바탕 / 프리텐다드
- 직접 웹폰트 CSS 넣어서 아무 폰트나 쓸 수 있음

### 📐 읽기 설정
- **글자 크기** 조절 (12~28px)
- **줄 간격** 조절 (1.2~3.0)
- **여백** 조절 (8~120px)
- 전부 슬라이더로 실시간 조절, 바로 보면서 맞출 수 있음

### 🔖 책갈피
- 헤더 북마크 아이콘으로 원터치 저장/삭제
- 저장된 페이지에 **캐릭터 초상화 리본** 표시
- 책갈피 목록에서 원하는 위치로 바로 점프

### 📚 판본 관리 (라이브러리)
- 캐릭터 초상화가 **책표지**로 나오는 그리드 뷰
- 여러 세션의 판본을 한눈에 관리
- 렌더링 결과 캐싱 → 다시 열 때 즉시 로드

### ✏️ 텍스트 편집
- 본문에서 텍스트 드래그 → **편집 팝오버**
- 원본 메시지 직접 수정 가능
- 수정하면 원본 채팅에도 반영됨

## 조작법

| 동작 | 방법 |
|------|------|
| 다음 페이지 | 우측 30% 터치 / 우측 키 / 스와이프 / 휠 |
| 이전 페이지 | 좌측 30% 터치 / 좌측 키 / 스와이프 / 휠 |
| 컨트롤 열기 | 화면 중앙 터치 |
| 책갈피 저장 | 상단 북마크 아이콘 (빈 아이콘일 때) |
| 책갈피 삭제 | 상단 북마크 아이콘 (꽉 찬 아이콘일 때) |
| 메뉴 | 하단 ☰ 버튼 |
| 텍스트 편집 | 본문 드래그 → 편집 버튼 |
| 리더 닫기 | 상단 ← 버튼 / ESC 키 |

---

## 설치 방법

1. `EbookReader.js` 파일을 다운로드
2. 리수AI → 설정 → 플러그인 → 플러그인 추가
3. 채팅 화면에서 📚 버튼 클릭
