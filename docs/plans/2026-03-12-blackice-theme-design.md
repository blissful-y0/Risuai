# Blackice Theme Design

**Date:** 2026-03-12

**Goal:** Create a dark companion theme to `dogeared` that preserves the same reader layout while shifting the mood to an ink-dark winter edition.

## Direction

`blackice` is the night edition of the existing literary reader theme:

- same bookmark / banner / header / body / action-bar layout
- same serif typography and prose rhythm
- darker paper, colder chrome, softer off-white text
- mobile banner that fades into dark paper instead of pale paper

## Visual Rules

- Surfaces use deep blue-black paper tones instead of white paper.
- Body text stays high-contrast, but not pure white.
- Metadata remains serif and quiet; `Opus` stays a small gray edition label.
- Quotes, code, and blockquotes become low-contrast ink washes rather than bright cards.
- Mobile keeps the full-bleed cover image treatment, but its fade resolves into darkness.

## Constraints

- Required Risu tags remain unchanged:
  - `<risutextbox>`
  - `<risuicon>`
  - `<risubuttons>`
  - `<risugeninfo>`
- Layout parity with `dogeared` is intentional so both themes can be tuned as light/dark counterparts.
